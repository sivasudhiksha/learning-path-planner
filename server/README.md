The Learning Path Planner follows a client-server architecture. The React frontend manages user interactions and communicates with the backend through REST APIs. The Node.js and Express backend processes requests, applies business logic, and interacts with MongoDB to store and retrieve user, learning path, and progress data.

ğŸ—„ï¸ Database Design

The database is designed to efficiently manage users, learning paths, modules, and progress tracking.

Entities:

User: Stores user authentication and profile details

LearningPath: Represents a structured learning roadmap

Module: Individual learning units within a learning path

Progress: Tracks user completion status of modules

Relationships are established to associate users with learning paths and track progress across modules.
